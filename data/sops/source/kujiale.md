## 第一阶段：接单（到此为止）

### 环节 1：群内发现订单

- **名称**：群内发现订单
- **自动化**：希望自动 — 监控指定群的客服消息，提取订单信息
- **位置**：企业微信 → 各企业接单群（仅监控标记为"接单群"的群）
- **触发**：客服在接单群发消息
- **做什么**：
  1. 客服发订单需求（含价格说明）
  2. 判断能不能接（看价格、看要求）
  3. 能接 → 私聊客服

---

### 环节 2：私聊 — 表达意向 + 确认接单

- **名称**：私聊接单
- **自动化**：希望自动 — 自动发"1"、转发群消息、发"接"
- **标准操作**（按顺序）：
  1. 给客服私聊发数字 **"1"**（表示有意向）
  2. **先问客服**：客户大致有什么要求？
  3. 客服回复后，觉得能做 → 发 **"接"** 确认接单
  4. 客服有时会要：之前合作过的**客户名字 + 作品案例图片**
  5. 提供后，客服会**拉一个新群**（你 + 客服 + 客户），开始沟通细节

- **输入**：群消息（订单描述 + 价格） + 客服补充的要求
- **输出**：新群（与客户沟通细节）

## 自动化方案调研（2026-06-26，二次更新）

### ⚠️ 关键前提：企微官方机器人无法用于此场景

经过深度调研确认：**企业微信官方所有机器人（群 Webhook 机器人、智能机器人），完全不支持外部群消息接收、不支持私聊微信个人用户。** 这是官方底层机制限制，无法通过任何配置突破。

你的酷家乐接单群属于"外部混合群"（含微信个人用户），所以**必须走非官方方案**。

### 方案一：QiweAPI（新发现，重点推荐）

**官网**：https://www.qiweapi.com

- **原理**：基于 RPA 技术的第三方 API 服务。你在企业微信里装一个"插件节点"，QiweAPI 通过这个节点操控企微——本质上也是模拟人手操作，但封装成了标准 HTTP API。
- **核心能力**：
  - ✅ **接收外部群消息**（回调 webhook 推送到凝华）— 这是官方做不到的
  - ✅ 发送群消息 / 私聊消息
  - ✅ 支持文本、图片、文件、视频等多种消息类型
  - ✅ 群管理（创建群、加减成员）
  - ✅ 事件回调（消息事件、登录状态等）
- **部署方式**：需要在一个设备上登录企微作为"节点"（可以是安卓手机或 Windows 电脑），QiweAPI 通过这个节点收发消息
- **收费**：官网写"免费试用"，商业使用需联系客服，具体价格不透明
- **优点**：API 完善、文档清晰、2026 年仍在活跃更新和维护
- **缺点**：商业产品价格不明、依赖第三方服务、消息经过第三方中转
- **API 示例**（伪代码）：
  ```
  收到群消息 → POST 回调到凝华
  凝华处理 → 发私聊: POST /v1/send {to: "客服昵称", type: "text", content: "1"}
  ```

### 方案二：iPad 协议（业内 SCRM 通用方案）

- **原理**：逆向你登录企业微信时 iPad 客户端和服务器之间的私有通信协议，然后用纯代码模拟 iPad 设备登录企微。不需要任何真实手机或电脑挂机，直接在 Linux 云端跑。
- **核心能力**：
  - ✅ 完美接收外部群 + 私聊所有消息
  - ✅ 发送文本/图片/文件/语音
  - ✅ 云端部署，7×24 小时运行，不占本地设备
  - ✅ 单机支持多账号并发
  - ✅ 可以和你现有的手机/电脑共存，不会挤掉线
- **部署**：Linux 服务器（可以就是你的开发机），纯代码运行
- **成本**：
  - 技术本身是公开的逆向知识，但有学习门槛
  - 市面上的"iPad 协议 SDK"通常是闭源商业产品，按月/年收费
  - 也有开源实现（如 wechat-bot/wecom-bot 等，但需要甄别质量）
- **优点**：稳定、不占设备、功能最强、业内 SCRM 首选方案
- **缺点**：非官方协议（企微大版本更新时需重新适配）、市面 SDK 良莠不齐、有账号风控风险

---

### 方案三：FlowBot / WorkBot（已基本否决）

**为什么降级**：

1. **两年没更新**：FlowBot GitHub 最后提交是 2024 年 6 月，WorkBot 也是。企微在两年间已经更新了多个大版本，兼容性存疑。
2. **收费不透明**：两者都声明"非商业用途免费"，你的酷家乐接单属于商业用途。想付费都不知道找谁、付多少。
3. **WorkBot 是 FlowBot 的 fork**：本质同一个东西，不存在"货比两家"。

**如果还是想试**：下载 WorkBot APK 在闲置安卓手机上跑免费版，先验证能不能收到接单群消息。免费版功能是否受限未知。

---

### 方案四：pywinauto（PC 桌面自动化，纯本地免费方案）

- **原理**：Python 控制你 Windows 电脑上的企业微信客户端窗口，模拟鼠标点击和键盘输入
- **优点**：完全免费、数据不出本地、不依赖任何第三方
- **缺点**：
  - 需要自己写自动化脚本（找到窗口、定位聊天列表、读取消息、点击发送）
  - 企微版本更新可能导致控件定位失效
  - 电脑不能关机/锁屏
  - 没有现成的消息回调机制，需要轮询窗口内容
- **适合**：如果你对 FlowBot/QiweAPI 都不满意，这是最后的备选
